## Why

Every session re-derived the same facts because the structured hook-bank (activeContext/progress/patterns) was machine-local and disconnected from the manager CLI (the M11 path split). The fleet standardized on symptom-indexed markdown notes + memgrep (janitor #16 / #11), which is strictly better for recall and sidesteps the privacy footgun (live memory never enters git). Keeping the old bank alongside it would be two memory systems with one stale — so it is retired, not repaired.
